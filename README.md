# Symfony API CRM

## Uruchomienie lokalne

```bash
# Uruchom środowisko
docker-compose up -d

# Sprawdź status
docker-compose ps

# Zatrzymaj środowisko
docker-compose down
```

## Dostęp

- Aplikacja: http://localhost
- MySQL: localhost:3306
- Redis: localhost:6379

## Dane dostępowe

- MySQL: symfony/symfony/symfony

## Serwer webowy

- Apache 2.4 z mod_php (PHP 8.3)
